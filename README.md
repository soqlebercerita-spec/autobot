## 🛠️ Installation

### Windows (Recommended)
```batch
# Clone repository
git clone https://github.com/yourusername/auratrade-bot.git
cd auratrade-bot

# Install dependencies
install_requirements.bat

# Start trading
start_bot.bat
```

### Manual Installation
```bash
# Install Python 3.11+
pip install -r requirements.txt

# Run bot
python bot.py
```

## 🚀 Quick Start

### 1. System Check
```batch
check_system.bat
```

### 2. Choose Trading Mode

**Safe Simulation Mode:**

**HFT Mode (Advanced):**
